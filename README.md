# SALAT Emilia Al-ZAraraee
SQL database planning- project.

## About the project
[SALAT] is a database design for an application of the following type: an application that shows prayer times to the user. The user can choose which prayers they want notifications for and can mute the others if desired. The application tracks the user's location and provides prayer time notifications based on their location.
